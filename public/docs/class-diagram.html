Связи между классами

Cart ---[1:*]---> CartItem (агрегация: корзина содержит товары)
CartItem ---[1:1]---> Book (ассоциация: товар корзины ссылается на книгу)
Order ---[1:*]---> CartItem (агрегация: заказ содержит товары)
Order ---[*:1]---> User (ассоциация: заказ принадлежит пользователю)
CatalogController ----uses----> Book (использует для фильтрации и сортировки)
CartController ----uses----> Cart (управляет корзиной)
UserController ----uses----> User, UserCredentials, UserRegistration
OrderController ----uses----> Order, OrderForm
All Controllers ----extends----> BaseController (наследование)
All Controllers ----uses----> ApiService, StorageService (зависимость)
